Structural directives

- *ngFor - *ngIf

*ngIf fait apparaître l’élément si l’expression introduite est vrai. Dans le cas contraire, elle supprime complètement l’élément.

La suppression est justifiée si, par exemple, vous devez afficher un component selon une condition bien précise. Si la condition n’est pas justifiée, le component n’est tout simplement pas activé et cela économisera des ressources.

Pour faire un Else if faut utiliser le ng template que vous allez voir si dessous

Explications

Dans l’ordre, voici ce qui a été appliqué : 1. Nous créons une balise « ng-template » sur laquelle nous lui assignons une « template reference variable ». Son nom : « elseBlock » (vous pouvez lui donner n’importe quel nom). 2. Au sein de notre « *ngIf », nous ajoutons un point-virgule suivi de l’instruction « else » ainsi que du nom « elseBlock ».
