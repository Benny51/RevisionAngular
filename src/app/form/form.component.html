Que cela soit pour ajouter, supprimer, modifier ou encore lire des données pour former un véritable CRUD, nous pouvons dire que les formulaires sont tout simplement omniprésents dans les applications ou sites web.
Création du premier type de formulaire
Pour que le formulaire fonctionne bien, il faut rajouter la directive ngForm dans la template reference variable qui se trouve à l'intérieur de la balise form.
Dans le cas où vous souhaiteriez voir si un champ précis est valide, il faut rajouter, toujours au sein de ce champ, un template reference variable qui vaudra ngModel.
