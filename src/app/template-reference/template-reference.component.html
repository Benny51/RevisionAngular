Avec le template référence, je consulte ici la couleur du p qui est aqua

{{nom_de_la_variable.style.color}} --> On récupère la couleur

{{nom_de_la_variable.textContent}} --> On récupère le texte

Par défaut, la valeur d’un template reference variable sera le DOM de l’élément sur lequel il est attaché. Cependant, nous pouvons altérer cela avec une directive13. La dernière chose à analyser concerne leur accès dans le component. Pour obtenir un lien dans la class de notre component, il faudra créer un attribut avec le décorateur @ViewChild(‘nom_de_la_variable’). Son accès, par contre, sera totalement garanti dès lors que le template se sera totalement chargé. Nous utiliserons donc ngViewAfterInit.
